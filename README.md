# 白菜杯 · 随机英雄 v2

30 人登录 · 10 人同时选英雄 · 极地大乱斗（ARAM）规则

## 功能

- **30 名队员** + **管理员** 独立登录
- 管理员创建 **两队对阵**（各 5 人），开启后 **10 人同时选英雄**
- 每人 **随机 3 选 1**，全局英雄 **不重复**
- 确认选将后，**未选的 2 个** 进入 **本方待选池**（队友可直接选用）
- **仅可见队友**已选英雄；敌方选将 **全员完成后** 才公开
- 可 **多次随机**，仅确认选将时才锁定并贡献待选池

## 本地运行

```bash
npm install
npm start
```

打开 http://localhost:8765

## 部署（Render）

仓库含 `render.yaml`，使用 **Node Web Service**（非静态站点）。

推送后 Render 自动部署，或 Dashboard 手动 Deploy latest commit。

## 使用流程

1. **管理员** 登录 → 选择 A/B 两队 → 创建对阵 → 开始选英雄
2. **10 名上场队员** 登录 → 同时随机/选将/从待选池抢英雄
3. **未上场队员** 可登录但仅见等待提示
4. 全员选完后公开双方阵容
